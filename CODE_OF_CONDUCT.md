# Code of Conduct

This project is intended to be open, practical, and respectful.

## Expected Behavior

- Be concise and constructive.
- Focus on the code, docs, and product behavior.
- Assume good intent unless there is evidence otherwise.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Deliberate sabotage of the codebase or release process
- Repeatedly ignoring maintainers' or contributors' boundaries

## Enforcement

Maintainers may remove comments, close issues, or decline contributions that violate this code of conduct.

If you need to report a problem, use GitHub Issues or a private security advisory when the problem involves sensitive details.

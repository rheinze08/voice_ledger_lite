What You Get

  • Local Room database for note storage
  • Title, body, created timestamp, and updated timestamp on every note
  • Cached semantic rollups so the app stays useful offline after a refresh
  • Ollama host and model settings in-app, defaulting to gemma4:e2b

Install Notes

  • This download is an Android .apk, not a Windows .exe.
  • It is currently a debug build for quick testing, not a Play Store signed release.
  • On Android, you may need to allow installs from your browser or file manager.
  • For emulator use, the default Ollama host is http://10.0.2.2:11434.

If The APK Link Fails

  1. Open the Build Android APK workflow.
  2. Click Run workflow.
  3. Leave publish_release enabled if you want the release and direct download link updated automatically.
  4. After the run finishes, refresh this page and the APK button will point to the latest uploaded asset.

Phone-local notes, summaries, and ask.

Voice Ledger Lite is an Android app for capturing notes locally, generating daily to yearly summary documents on-device, and querying your own data without standing up a separate server.

Latest Release Repository
What to expect on first launch: the app opens to a setup screen and downloads its local AI models into app-private storage. Leave it open on Wi-Fi. The summary model is large and dominates the first-run download.

Setup

The app checks and downloads the summary and embedding models automatically. This is where the user should expect the large initial download.

  • Shows per-model install state
  • Shows download progress and byte counts
  • Allows retry if the network drops

Notes

The Notes screen separates the currently active note from older records so you can pull a previous record back into focus quickly.

  • Active section for the selected note
  • Records list for prior notes
  • Fast jump from historical records back into the active view

Compose

Capture a note with a title, editable date, body text, and reusable tags. The date defaults to today but can be changed for backfilled notes.

  • Manual date field in YYYY-MM-DD
  • Up to five reusable tags
  • One note can carry multiple tags

Ask

Ask runs retrieval over notes and summary documents, then uses the local model to synthesize an answer from the targeted context.

  • Semantic narrowing over your own data
  • Final answer generated from retrieved documents
  • Useful for question answering rather than raw note browsing

Summarize

Summarize is where the app updates checkpoints, generates daily/weekly/monthly/yearly summary documents, and exposes model installation status.

  • Update for incremental refresh
  • Rebuild for replay from the configured floor
  • Summary documents grouped by time period
